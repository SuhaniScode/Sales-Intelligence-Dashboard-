# 📊 Sales Intelligence Dashboard

## 🚀 Overview
I built this project to transform raw sales data into meaningful business insights.

Instead of just looking at numbers, this dashboard helps answer:
- Who are the top customers?
- Which categories drive the most revenue?
- Where is profit coming from?

---

## 📸 Dashboard Preview
![Dashboard](Dashboard.png)

---

## 🧠 Problem
Raw data is messy and hard to understand.  
Businesses need **clear insights**, not spreadsheets.

---

## 💡 Solution
I created an interactive dashboard that converts raw data into:
- Actionable insights  
- Visual understanding  
- Better decision-making  

---

## 🛠️ Tech Stack
- **SQL (SQLite)** → Data analysis & joins  
- **Power BI** → Dashboard & visualization  
- **Excel** → Data cleaning & preparation  

---

## 📊 Key Insights
- Technology category generates highest sales  
- West region leads in profit  
- Top customers contribute significantly to revenue  
- Both profit and loss patterns exist in transactions  

---

## ⚡ Features
- Interactive dashboard with slicers  
- Top N customer analysis  
- Category-wise sales breakdown  
- Region-wise profit analysis  
- Clean data modeling (Sales, Customer, Product)

- SQL is powerful for extracting insights
